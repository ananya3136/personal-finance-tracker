# Personal Finance AI Tracker

An AI-powered financial analytics platform built with MERN stack.

## Features
- Budget tracking
- Anomaly detection
- Spending prediction
- Financial health scoring
- AI financial strategy generation

## Tech Stack
- Node.js
- Express
- MongoDB
- React
- JWT Authentication
- Ollama (Llama 3)

## Architecture
Client → API → Business Logic → MongoDB → AI Layer

## How to Run
### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
